# Portfolio Template

This repository now includes a simple one-page portfolio website for the data collection professional.

## Files
- [index.html](index.html) — main portfolio content and layout
- [styles.css](styles.css) — styling and responsive design

## How to use
1. Open [index.html](index.html) in your browser to preview the page.
2. Replace the two placeholder image areas with your actual project photos.
3. Update the contact email and any wording you want to personalize.

## Optional preview
You can also run a local web server from the repository folder with:

```bash
python3 -m http.server
```

Then visit `http://localhost:8000` in your browser.
